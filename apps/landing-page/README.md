### Setup env

```bash
  cp .env.example .env
```

### Lint the files
```bash
pnpm run lint
```

### Start development
```bash
pnpm run dev
```

### Build Prod
```bash
pnpm run build
```

### Techstacks
- **Astro** - Static site generator
- **Vue.js 3** - Component framework
- **Tailwind CSS** - Utility-first CSS framework
- **Pinia** - State management
- **TypeScript** - Type safety
